ROBO-SUMM

An audio/video transcriptor


ABOUT


Robo-Summ is an audio/video transcriber built using AssemblyAI's free transcription API for DeltaHacks 8 by Huzaifa Zia, Ryan Loftus and Zia Formuly. It can be used to transcribe any audio or video link into a neatly generated rich text format and output results into neatly divided sections.
